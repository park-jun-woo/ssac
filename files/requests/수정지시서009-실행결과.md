# 수정지시서 009 실행결과: @response 템플릿에서 HTTP 상태 코드 제거

## 실행일: 2026-03-13

## 방침

`@response` 성공 응답의 HTTP 상태 코드는 OpenAPI가 SSOT. SSaC는 `http.StatusOK` 대신 `__RESPONSE_STATUS__` 마커를 출력하고, fullend가 OpenAPI를 보고 치환.

## 변경 내용

### 1. `generator/go_templates.go` — response 템플릿 2곳

- `response`: `c.JSON(http.StatusOK, gin.H{...})` → `c.JSON(__RESPONSE_STATUS__, gin.H{...})`
- `response_direct`: `c.JSON(http.StatusOK, {{.Target}})` → `c.JSON(__RESPONSE_STATUS__, {{.Target}})`

### 2. `generator/go_handler_test.go` — assertion 업데이트

- `TestGenerateResponseDirect`: `http.StatusOK` → `__RESPONSE_STATUS__` 검증

## 변경 파일

| 파일 | 변경 |
|---|---|
| `generator/go_templates.go` | `response`, `response_direct` 템플릿에서 `http.StatusOK` → `__RESPONSE_STATUS__` |
| `generator/go_handler_test.go` | 2개 assertion 업데이트 |

## 영향 없는 범위

- `@empty` 404, `@exists` 409, `@state` 409, `@auth` 403 — 그대로 유지 (SSaC 소관)
- `@call` ErrStatus — 그대로 유지
- `@publish` 500 — 그대로 유지
