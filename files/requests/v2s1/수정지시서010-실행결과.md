## 변경 내용

### generator/go_target.go

`inputValueToCode()` 수정 — 모델 result 변수의 필드 참조 시 `*` 역참조 접두사 생성:

```go
func inputValueToCode(val string) string {
    if val == "query" { return "opts" }
    if strings.HasPrefix(val, "request.") { return lcFirst(val[len("request."):]) }
    if strings.HasPrefix(val, "currentUser.") || strings.HasPrefix(val, "config.") { return val }
    if strings.HasPrefix(val, `"`) { return val }  // 리터럴
    if strings.ContainsRune(val, '.') { return "*" + val }  // 모델 필드 → 역참조
    return val  // bare variable
}
```

NOT NULL 여부를 구분하지 않고 모델 필드를 전부 포인터로 통일 취급. DB가 NOT NULL 검증 담당.
