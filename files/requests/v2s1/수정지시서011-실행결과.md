# 수정지시서 011 실행결과: @call 결과 변수 필드는 역참조하지 않음

## 실행일: 2026-03-10

## 변경 내용

### generator/go_target.go

`inputValueToCode()`에 `callResultVars map[string]bool` 파라미터 추가. `source.Field`에서 source가 @call 결과 변수이면 역참조하지 않음:

```go
func inputValueToCode(val string, callResultVars map[string]bool) string {
    // ...
    if dotIdx := strings.IndexByte(val, '.'); dotIdx > 0 {
        source := val[:dotIdx]
        if callResultVars[source] { return val }  // @call 결과 → 역참조 안 함
        return "*" + val  // 모델 결과 → 역참조
    }
    // ...
}
```

`GenerateFunc()`에서 @call 결과 변수 수집:

```go
callResultVars := map[string]bool{}
for _, seq := range sf.Sequences {
    if seq.Type == parser.SeqCall && seq.Result != nil {
        callResultVars[seq.Result.Var] = true
    }
}
```

`callResultVars`를 `buildTemplateData()` → `buildArgsCodeFromInputs()`, `buildInputFieldsFromMap()` → `inputValueToCode()`로 전달.

### generator/generator_test.go

`TestGenerateCallResultFieldNoDeref` 추가:

```go
// @call auth.HashPasswordResponse hashedPassword = auth.HashPassword({Password: request.Password})
// @post User user = User.Create({PasswordHash: hashedPassword.HashedPassword, ...})
// → hashedPassword.HashedPassword (역참조 없음)
// → *hashedPassword.HashedPassword 가 아님
```

## 역참조 규칙 정리

| source | 역참조 | 이유 |
|---|---|---|
| `reservation.Status` | `*reservation.Status` | @get 모델 결과 → 포인터 필드 |
| `hashedPassword.HashedPassword` | `hashedPassword.HashedPassword` | @call 결과 → Response struct 일반 필드 |
| `request.CourseID` | `courseID` (변환) | HTTP 파라미터 → 값 타입 |
| `currentUser.ID` | `currentUser.ID` | 인증 컨텍스트 → 값 타입 |
| `"cancelled"` | `"cancelled"` | 리터럴 |
