# 수정지시서 023 실행결과: 미사용 변수 _ 시 := → = 전환 + config.Get 타입 변환

## 실행일: 2026-03-11

## 변경 내용

### A. 미사용 변수 _ 시 := → = 전환

#### generator/go_target.go

`templateData`에 `ErrDeclared bool` 필드 추가:

```go
type templateData struct {
    // ...
    Unused      bool
    ErrDeclared bool // err variable already declared before this sequence
}
```

`buildTemplateData()`에서 err 추적 블록 직전에 현재 `*errDeclared` 상태 캡처:

```go
// capture errDeclared state before this sequence modifies it
d.ErrDeclared = *errDeclared
```

`generateHTTPFunc()`/`generateSubscribeFunc()`에서 Unused 설정 시 ErrDeclared면 ReAssign 강제:

```go
if seq.Result != nil && !usedVars[seq.Result.Var] {
    data.Unused = true
    if data.ErrDeclared {
        data.ReAssign = true // _, err = (no new vars with :=)
    }
}
```

결과:
- `_` + err 첫 선언 → `_, err :=` (기존 동작 유지)
- `_` + err 이미 선언 → `_, err =` (수정: Go 컴파일 에러 해결)

#### generator/go_templates.go

변경 없음. 기존 `{{if .ReAssign}}={{else}}:={{end}}` 로직이 ReAssign 오버라이드로 정확히 동작.

### B. config.Get 타입 변환

#### generator/go_target.go

`inputValueToCode(val string)` → `inputValueToCode(val string, targetType string)` 시그니처 변경:

```go
func inputValueToCode(val string, targetType string) string {
    // ...
    if strings.HasPrefix(val, "config.") {
        key := val[len("config."):]
        upperKey := toUpperSnake(key)
        switch targetType {
        case "int":
            return `config.GetInt("` + upperKey + `")`
        case "int32":
            return `int32(config.GetInt("` + upperKey + `"))`
        case "int64":
            return `config.GetInt64("` + upperKey + `")`
        case "bool":
            return `config.GetBool("` + upperKey + `")`
        default:
            return `config.Get("` + upperKey + `")`
        }
    }
}
```

`buildInputFieldsFromMap(inputs, paramTypes)` — `paramTypes map[string]string` 파라미터 추가. @call 시 `MethodInfo.ParamTypes` 전달, 그 외(state, auth)는 `nil`.

`buildArgsCodeFromInputs()`, `buildPublishPayload()` — targetType `""` 전달 (기본 string).

`lookupCallParamTypes()` 헬퍼 추가: @call 모델에서 심볼 테이블의 ParamTypes 조회.

`buildTemplateData()`의 @call 분기에서 ParamTypes 조회하여 `buildInputFieldsFromMap` 전달:

```go
if seq.Type == parser.SeqCall && st != nil && seq.Model != "" {
    paramTypes = lookupCallParamTypes(seq.Model, st)
}
d.InputFields = buildInputFieldsFromMap(seq.Inputs, paramTypes)
```
