# 수정지시서 004 실행결과: CanTransition 반환 타입 bool → error

## 실행일: 2026-03-10

## 변경 내용

### generator/go_templates.go

state 템플릿 변경:

```go
// 변경 전
if !{{.DiagramID}}state.CanTransition({{.DiagramID}}state.Input{...}, "{{.Transition}}") {
    c.JSON(http.StatusConflict, gin.H{"error": "{{.Message}}"})

// 변경 후
if err := {{.DiagramID}}state.CanTransition({{.DiagramID}}state.Input{...}, "{{.Transition}}"); err != nil {
    c.JSON(http.StatusConflict, gin.H{"error": err.Error()})
```

- `!bool` → `err != nil` 패턴
- 에러 메시지: SSaC 선언의 고정 Message → `err.Error()` (상태 머신이 구체적 사유 제공)
