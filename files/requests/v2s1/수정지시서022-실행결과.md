## 실행일: 2026-03-11

## 변경 내용

### A. @auth → @call 방식 코드 생성

#### generator/go_templates.go

`auth`/`sub_auth` 템플릿을 `@call` 방식으로 변경:

```go
// 변경 전:
if err := authz.Check(currentUser, "action", "resource", authz.Input{...}); err != nil {

// 변경 후:
if _, err := authz.Check(authz.CheckRequest{Action: "action", Resource: "resource", ...}); err != nil {
```

- `auth` 템플릿: `authz.Check(authz.CheckRequest{Action: ..., Resource: ..., InputFields})` + 403 Forbidden
- `sub_auth` 템플릿: 동일 패턴 + `return fmt.Errorf(...)` 에러

#### generator/go_target.go

`needsCurrentUser()`: `seq.Type == parser.SeqAuth` 무조건 true 제거. @auth inputs에 `currentUser.X`가 있으면 기존 Inputs 체크로 감지됨.

### B. @call 입력 타입 검증

#### validator/symbol.go

`MethodInfo`에 `ParamTypes map[string]string` 추가:

```go
type MethodInfo struct {
    Cardinality string
    Params      []string
    ParamTypes  map[string]string // 파라미터명 → Go 타입
}
```

`loadPackageGoInterfaces()` 확장:
1. 1차 패스: `{Method}Request` struct를 파싱하여 필드명→타입 매핑 수집
2. 2차 패스: interface 메서드 파싱 시 `ParamTypes`에 Request struct 필드 매핑

`exprToGoType()` 헬퍼 추가: `ast.Expr` → Go 타입 문자열 변환.

#### validator/validator.go

`validateCallInputTypes()` 추가:
- @call inputs의 각 값에서 타입 결정 (DDL 역추적)
- Request struct `ParamTypes`와 비교
- 불일치 시 ERROR: `입력 "Amount"의 타입 불일치: gig.Budget은 int64, Request 필드는 int`

`resolveCallInputType()` 헬퍼: 변수.Field → DDL 테이블 컬럼 타입 역추적.

### C. 미사용 변수 _ 처리

#### generator/go_target.go

`templateData`에 `Unused bool` 필드 추가.

`collectUsedVars()` 함수 추가: 시퀀스 루프 전에 참조 분석 (Guard Target, Inputs values, Response Fields).

`generateHTTPFunc()`/`generateSubscribeFunc()`에서 `usedVars` 사전 스캔 후, result가 있고 `usedVars[result.Var] == false`이면 `data.Unused = true`.

#### generator/go_templates.go

`get`, `post`, `call_with_result`, `sub_get`, `sub_post`, `sub_call_with_result` 템플릿에 Unused 분기 추가:

```go
{{if .Unused}}_{{else}}{{.Result.Var}}{{end}}, err :=
```

### D. config.* → config.Get() 코드젠 변환

#### generator/go_target.go

`inputValueToCode()` 변경:

```go
if strings.HasPrefix(val, "config.") {
    key := val[len("config."):]
    return `config.Get("` + toUpperSnake(key) + `")`
}
```

`toUpperSnake()` 헬퍼 추가: `toSnakeCase()` + `strings.ToUpper()`.

`needsConfig()` 헬퍼 추가: 시퀀스 inputs에 `config.` 참조 존재 확인.

`collectImports()`/`collectSubscribeImports()`에서 `needsConfig()` 시 `"config"` import 추가.
