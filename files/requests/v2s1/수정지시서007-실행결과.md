# 수정지시서 007 실행결과: @call args를 JSON 형태 named field로 변경

## 실행일: 2026-03-10

## 변경 내용

### parser/parser.go

`parseCall()` → `parseCallExprInputs()` 사용. `({Key: val, ...})` → `seq.Inputs` map에 저장.

```go
// 변경 전
model, args := parseCallExpr(rhs)
seq.Model = model
seq.Args = args

// 변경 후
model, inputs := parseCallExprInputs(rhs)
seq.Model = model
seq.Inputs = inputs
```

`parseCallExprInputs()` 추가: `(` 와 `)` 사이를 `parseInputs()`로 파싱.

### generator/go_target.go

- `buildTemplateData()`: `@call` 전용 `buildCallInputFields(seq.Args)` 오버라이드 제거. `seq.Inputs` → `buildInputFieldsFromMap()` 공통 경로 사용.
- `buildCallInputFields()`, `callFieldName()` 삭제.
