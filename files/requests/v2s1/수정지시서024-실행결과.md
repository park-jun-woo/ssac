# 수정지시서 024 실행결과: config.* 지원 전면 제거

## 실행일: 2026-03-11

## 변경 내용

### validator/validator.go

#### 1. declared에서 config 제거

```go
// 변경 전
declared := map[string]bool{"currentUser": true, "config": true}

// 변경 후
declared := map[string]bool{"currentUser": true}
```

#### 2. config.* 사용 시 ERROR 추가 + 허용 제거

`validateVariableFlow()` Inputs 순회에서:
- `config.*` 감지 시 ERROR: `config.SMTPHost — config.* 입력은 지원하지 않습니다. func 내부에서 os.Getenv()를 사용하세요`
- ref 검증에서 `ref != "config"` 조건 제거

#### 3. resolveCallInputType()에서 config 분기 삭제

```go
// 삭제: if strings.HasPrefix(val, "config.") { return "string" }
```

#### 4. validateConfigTypes() 전체 삭제

`configSupportedTypes` 변수 + `validateConfigTypes()` 함수 전체 삭제.
`ValidateWithSymbols()`에서 호출 제거.

#### 5. reservedSources는 유지

`"config": true` 유지 — result 변수명으로 `config` 사용 금지는 그대로.

### generator/go_target.go

#### 1. inputValueToCode() 단순화

```go
// 변경 전: func inputValueToCode(val string, targetType string) string { ... config 블록 ... }
// 변경 후: func inputValueToCode(val string) string { ... config 블록 삭제 ... }
```

모든 호출부에서 `targetType` 인자 제거:
- `buildInputFieldsFromMap(inputs)` — `paramTypes` 파라미터 제거
- `buildArgsCodeFromInputs()` — `inputValueToCode(inputs[k])` 호출
- `buildPublishPayload()` — 동일

#### 2. 삭제된 함수/변수

- `needsConfig()` — config import 감지
- `lookupCallParamTypes()` — @call ParamTypes 조회
- `toUpperSnake()` — PascalCase→UPPER_SNAKE_CASE 변환

#### 3. collectImports()/collectSubscribeImports()

config import 분기 삭제: `if needsConfig(...) { seen["config"] = true }` 제거.

#### 4. argToCode()

`a.Source == "config"` 분기 제거: `if a.Source == "currentUser" || a.Source == "config"` → `if a.Source == "currentUser"`

#### 5. collectUsedVars()

`strings.HasPrefix(val, "config.")` 분기 삭제.

#### 6. buildTemplateData() @call 분기

`lookupCallParamTypes` 호출 제거, `buildInputFieldsFromMap(seq.Inputs)` 단순 호출로 변경.
