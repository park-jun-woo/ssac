# 수정지시서 012: 모델 필드 역참조(`*`) 원복

## 배경

수정지시서010에서 모델 필드 참조 시 `*` 역참조를 추가했고, 수정지시서011에서 @call 결과를 제외했다.

방침 변경: nullable 컬럼을 `NOT NULL DEFAULT 0`으로 바꿔서 포인터 타입 자체를 없앤다. 따라서 역참조 로직이 불필요.

## 변경 사항

수정지시서010, 011에서 추가한 역참조 로직을 전부 원복한다:

1. `inputValueToCode()`에서 `"*" + val` 역참조 제거 → 원래대로 `val` 반환
2. `callResultVars` 파라미터 및 관련 로직 제거
3. 테스트도 원복 (`*course.ID` → `course.ID` 등)

## 기대 동작

모든 `source.field` 참조가 역참조 없이 그대로 생성된다:
- `gig.FreelancerID` (O)
- `*gig.FreelancerID` (X)
