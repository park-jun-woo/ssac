# 수정지시서 024: config.* 지원 전면 제거

## 배경

`config.*` 입력값은 SSaC 코드젠이 Go 타입 시스템을 추적해야 하는 복잡성을 유발한다 (수정지시서023 B의 config 타입 변환, @func 패키지 ParamTypes 등록 문제 등).

설계 변경: 인프라 설정(SMTP, DB 등)은 func 내부에서 직접 `os.Getenv()`로 읽는다.
SSaC @call은 비즈니스 데이터만 전달한다.

## 변경 목표

`config.*` 값을 SSaC inputs에서 사용하면 **validator ERROR**로 거부한다.
generator에서 config 관련 코드젠을 모두 제거한다.
