# 수정지시서 023: 미사용 변수 _ 시 := → = 전환 + config.Get 타입 변환

## 현재 문제

1. 미사용 변수 `_` 처리 시 `_, err :=` 생성 → `err`이 이미 선언된 경우 Go `no new variables on left side of :=` 에러.
   `_`는 blank identifier이므로 새 변수가 아님 → `:=` 는 최소 하나의 새 변수가 필요.

2. `config.Get()` 은 항상 `string` 반환. func Request 필드가 `int`, `map[string]string` 등일 때 타입 불일치 컴파일 에러.

## 변경 목표

### A. 미사용 변수 _ 시 := → = 전환

`_` 결과 변수 시:
- `err`이 이전 시퀀스에서 이미 선언되었으면 `_, err = func(...)` 생성
- `err`이 첫 선언이면 `_, err := func(...)` 생성

판단 기준: 해당 시퀀스 이전에 `err`을 사용하는 시퀀스가 하나라도 있으면 `err`은 이미 선언된 것.

### B. config.Get 타입 변환

`config.Key` → func Request 필드 타입에 따라 변환:

| Request 필드 타입 | 생성 코드 |
|---|---|
| `string` | `config.Get("KEY")` |
| `int` | `config.GetInt("KEY")` |
| `int32` | `int32(config.GetInt("KEY"))` |
| `int64` | `config.GetInt64("KEY")` |
| `bool` | `config.GetBool("KEY")` |
| 기타 (`map[string]string` 등) | ERROR: `config.Key는 타입 T로 변환할 수 없습니다` |

fullend `pkg/config`에 추가 필요 함수:
- `GetInt(key string) int`
- `GetInt64(key string) int64`
- `GetBool(key string) bool`

## 변경 내용

### generator/go_target.go — := vs = 판단

`generateHTTPFunc()`/`generateSubscribeFunc()`에서 Unused 분기 시:

현재:
```go
{{if .Unused}}_{{else}}{{.Result.Var}}{{end}}, err :=
```

변경 후:
```go
{{if .Unused}}_{{else}}{{.Result.Var}}{{end}}, err {{if .ErrDeclared}}={{else}}:={{end}}
```

`ErrDeclared bool` 필드 추가: 해당 시퀀스 이전에 `err`을 사용하는 시퀀스가 있으면 `true`.

### generator/go_target.go — config.Get 타입 변환

`inputValueToCode()`에서 `config.Key` 처리 시 func Request 필드 타입을 참조:

현재:
```go
if strings.HasPrefix(val, "config.") {
    key := val[len("config."):]
    return `config.Get("` + toUpperSnake(key) + `")`
}
```

변경 후:
```go
if strings.HasPrefix(val, "config.") {
    key := val[len("config."):]
    upperKey := toUpperSnake(key)
    switch targetType {
    case "int":
        return `config.GetInt("` + upperKey + `")`
    case "int32":
        return `int32(config.GetInt("` + upperKey + `"))`
    case "int64":
        return `config.GetInt64("` + upperKey + `")`
    case "bool":
        return `config.GetBool("` + upperKey + `")`
    default:
        return `config.Get("` + upperKey + `")`
    }
}
```

`inputValueToCode()`에 `targetType string` 파라미터 추가. 호출 시 func Request 필드 타입을 전달.

### validator/validator.go — config 타입 검증

`config.Key`가 func Request 필드에 대입될 때, 필드 타입이 변환 불가능한 타입(예: `map[string]string`)이면 ERROR.

## fullend 측 연동 변경 (별도)

`pkg/config`에 추가:

```go
func GetInt(key string) int {
    v := os.Getenv(key)
    if v == "" { return 0 }
    n, _ := strconv.Atoi(v)
    return n
}

func GetInt64(key string) int64 {
    v := os.Getenv(key)
    if v == "" { return 0 }
    n, _ := strconv.ParseInt(v, 10, 64)
    return n
}

func GetBool(key string) bool {
    v := os.Getenv(key)
    return v == "1" || v == "true" || v == "TRUE"
}
```
