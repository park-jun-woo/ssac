# 수정지시서 008 실행결과: CRUD args를 {Key: value} 문법으로 통일

## 실행일: 2026-03-10

## 변경 내용

### parser/parser.go

`parseCRUD()` → `parseCallExprInputs()` 사용. 기존 `parseCallExpr()` → `seq.Args` 대신 `({Key: val})` → `seq.Inputs` map에 저장.

```go
// 변경 전
model, args := parseCallExpr(rhs)
seq.Model = model
seq.Args = args

// 변경 후
model, inputs := parseCallExprInputs(rhs)
seq.Model = model
seq.Inputs = inputs
```

### generator/go_target.go

- `buildTemplateData()`: CRUD → `buildArgsCodeFromInputs(seq.Inputs)` (값만 추출, 키 알파벳순 정렬)
- `hasQueryInput(seq.Inputs)` → query 예약 소스 감지
- `modelUsage.Args` → `modelUsage.Inputs` map으로 변경
- `deriveInterfaces()`: Inputs 키를 파라미터 이름으로 사용
- `hasQueryArg()` 삭제, `resolveInputParamType()` 추가

### validator/validator.go

- @post/@put/@delete 필수 필드 검사: `len(seq.Args)` → `len(seq.Inputs)`
- `validateVariableFlow()`: Inputs에서 리터럴(`"..."`)과 예약 소스(`query`, `config`) 스킵
