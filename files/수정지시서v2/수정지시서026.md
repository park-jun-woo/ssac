# 수정지시서 026: @call 에러 HTTP 상태 코드 커스텀 지정 + 기본값 500

## 문제

`call_no_result` 템플릿이 에러 시 무조건 401 Unauthorized를 반환한다.
`mail.SendTemplateEmail` 같이 인증과 무관한 함수도 401이 되어 클라이언트가 오인한다.

## 해결 방향

1. `@call` 기본 에러 코드를 500으로 변경 (call_with_result과 통일)
2. `@call` 끝에 선택적으로 에러 코드를 지정할 수 있게 확장

```ssac
// 기본 (500)
// @call mail.SendTemplateEmail({To: user.Email, Subject: "Welcome"})

// 명시적 지정 (401)
// @call auth.VerifyPassword({Email: email, Password: password}) 401
```

옵셔널 파라미터는 끝에 위치한다.

## 수정 파일

### 1. parser/types.go — Sequence에 ErrStatus 필드 추가

```go
type Sequence struct {
    // ... 기존 필드 ...

    // 공통
    Message      string // 에러 메시지
    ErrStatus    int    // @call 에러 HTTP 상태 코드 (0이면 기본값 사용)
    SuppressWarn bool   // @type! — WARNING 억제
}
```

### 2. parser/parser.go — parseCall에서 trailing 숫자 파싱

`parseCall` 함수에서 `({...})` 뒤에 남은 텍스트가 3자리 숫자(HTTP status)이면 `seq.ErrStatus`에 저장.

```go
func parseCall(rest string) (*Sequence, error) {
    // ... 기존 result/model/inputs 파싱 ...

    // 끝에 trailing HTTP status code 파싱 (예: "401", "402")
    // parseCallExprInputs가 소비하지 않은 나머지에서 추출
    // → seq.ErrStatus = 401
    return seq, nil
}
```

구체적으로: `parseCallExprInputs`가 `(...)` 까지 소비한 후 남은 rest에서 숫자를 추출.
현재 `parseCallExprInputs`는 rest를 반환하지 않으므로, 반환값에 remainder를 추가하거나
별도로 원본 rest에서 닫는 괄호 뒤를 파싱.

### 4. generator/go_target.go — ErrStatus를 템플릿 데이터에 전달

`buildTemplateData` (또는 해당 함수)에서:

```go
// @call의 ErrStatus 결정
errStatus := "http.StatusInternalServerError" // 기본값 500
if seq.ErrStatus != 0 {
    errStatus = httpStatusConst(seq.ErrStatus) // 401 → "http.StatusUnauthorized"
}
d.ErrStatus = errStatus
```

`httpStatusConst` 헬퍼:
```go
func httpStatusConst(code int) string {
    switch code {
    case 400: return "http.StatusBadRequest"
    case 401: return "http.StatusUnauthorized"
    case 402: return "http.StatusPaymentRequired"
    case 403: return "http.StatusForbidden"
    case 404: return "http.StatusNotFound"
    case 409: return "http.StatusConflict"
    case 429: return "http.StatusTooManyRequests"
    case 502: return "http.StatusBadGateway"
    case 503: return "http.StatusServiceUnavailable"
    default:  return fmt.Sprintf("%d", code)
    }
}
```
